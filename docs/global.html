getChangeColor

Determines the color associated with a given percentage change value. - Green for positive changes - Red for negative changes - Gray for neutral or zero change
Source:
  • utils/utils.js, line 44
Examples
// Returns 'green' for a positive change
const color = getChangeColor('1.5');
// Returns 'red' for a negative change
const color = getChangeColor('-0.5');
// Returns 'gray' for a neutral change
const color = getChangeColor('0');
